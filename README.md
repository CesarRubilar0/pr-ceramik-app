# pr.ceramik

Sitio web de presentación para `pr.ceramik`, un estudio de cerámica surrealista. La página muestra piezas destacadas, una galería visual y datos de contacto para la marca.

## Qué es esta página

Esta es una landing page comercial simple pensada para exhibir piezas de arte en cerámica, con un enfoque en la estética, el proceso creativo y la conexión con compradores o coleccionistas. No es una tienda completa; es una página de presentación y portafolio.

## Para quién es

- Estudio de cerámica o artista ceramista
- Vendedores de piezas de arte para ferias y exposiciones
- Coleccionistas y clientes interesados en objetos decorativos únicos
- Personas que necesitan una presencia visual rápida en web sin e-commerce complejo

## Tipo de página

- Landing page / portafolio
- Página de presentación de producto
- Experiencia estática centrada en diseño y contenido visual

## Stack tecnológico

- `Vite` — bundler y servidor de desarrollo
- `React` — biblioteca UI para renderizar componentes
- `JavaScript` — lógica de la aplicación
- `HTML` y `CSS` — estructura y estilos
- `public/` — imágenes y recursos estáticos

## Estructura del proyecto

- `src/App.jsx` — componente principal de la app
- `src/data.js` — datos de piezas, reseñas y eventos
- `src/main.jsx` — punto de entrada de React
- `src/style.css` — estilos principales
- `public/images/` — imágenes usadas en la página

## Mejores siguientes pasos

1. Mejorar la galería con mejores imágenes y nombres reales.
2. Agregar navegación directa a secciones de la página.
3. Convertirlo en una landing responsiva más pulida en móviles.
4. Añadir un formulario de contacto o WhatsApp integrado.
5. Agregar animaciones sutiles o transiciones con CSS.
6. Se presentará el proyecto a Paola de pr.ceramik para evaluar convertirlo en un e-commerce con carrito.
7. Revisar qué comentarios o mejoras adicionales se necesitan aplicar.

## Despliegue

La app puede desplegarse fácilmente en Vercel. Solo hay que conectar el repositorio y usar el comando de build:

```bash
npm install
npm run build
```

## Enfoque de Desarrollo y Visión de Proyecto

Este proyecto está realizado de manera práctica y sencilla, buscando una fluidez que apoye directamente al posicionamiento de talleres locales y PYMEs en general, impulsando su transformación digital sin fricciones.

Este README documenta una primera versión funcional, pensada para crecer y adaptarse según las necesidades del estudio y su presencia online.
